updateAccount Method
Order Name Type
Return accountRef string
Input 1 account Account
Input 2 createFieldAndValues boolean

Comments
1) This method will update an existing account.
2) A soap fault will occur if the parameter account is empty or invalid.  Review Account to help avoid a soap fault.
3) This method will add or update the Account personaType provided depending on whether the account already has the personaType.  If the Account personaType is empty, the account's primary persona will be updated.
4) If the parameter createFieldAndValues is true, this indicates all DefinedValue fields and values that aren't located will be created.  When this parameter is set to true, don't forget to set each DefinedValue dataType and displayType to appropriate values.  Please note, the personaType will not be created if it is invalid.
