addContact Method
Order Name Type
Return contactRef string
Input 1 contact Contact
Input 2 createFieldAndValues boolean

Comments
1) This method will add a new journal contact to an existing account.
2) A soap fault will occur if the parameter contact is empty or invalid.  Review Contact to help avoid a soap fault.
3) If the parameter createFieldAndValues is true, the following items will be created if they aren't located: the Contact method and all DefinedValue fields and values.  When this parameter is set to true, don't forget to set each DefinedValue dataType and displayType to appropriate values.
