Errors
Num Short Description Comments
1 Unknown Error eTapestry encountered an unexpected error while processing the request. Contact api@etapestry.com for assistance.
2 Invalid Login The login id and/or password provided is invalid.
3 Missing Name An Account is missing a name or name parts (title, first, middle, and last).
4 Invalid Month A CreditCard has an invalid expirationMonth.  This int must be from 1 and 12.
5 Invalid Year A CreditCard has an invalid expirationYear.  This int must be from 1900 to 9999.
6 Field Not Found A DefinedValue has a fieldName that doesn't exist within the eTapestry database.
7 Duplicate Matches Found A DefinedValue has been located on multiple accounts.
8 Ref Not Found A database ref could not be located within the eTapestry database.
9 Invalid Persona Type An Account has a personaType that doesn't exist within the eTapestry database.
10 Invalid Fund A transaction has a fund name that doesn't exist within the eTapestry database.
11 Invalid Amount A transaction has an amount that is invalid.  Review each transaction to determine what is appropriate.
12 Missing Processor The organization is missing a transaction processor.
13 Credit Card Process Error The organization's transaction processor encountered an error while processing a credit card.  Review the fault string for further explanation.
14 EFT Process Error The organization's transaction processor encountered an error while processing eft valuable.  Review the fault string for further explanation.
15 Invalid License The organization does not have an active license to access the API.
16 Account Locked Out The login id's access has been temporarily locked out of eTapestry.
18 Password Length Error The length of the new password does not meet the organization's required length.
19 Password Login Error The new password cannot match the login id.
20 Password Value Error The new password cannot be 'password', consecutive characeters ('123456') or same characters ('111111').
21 Password Complex Error The new password must contain at least 3 of the following 4 items - a lower case letter, an upper case letter, a number, and a special character
22 Password History Error The new password cannot match any of last x number of passwords.  Each organization defines how many unique passwords are needed before an old one can be reused.
23 Password Generic Error eTapestry encountered an unexpected error while trying to save a new password. Contact api@etapestry.com for assistance.
24 Invalid Installment Amount A StandardPaymentSchedule has a installment amount that is invalid.
25 Invalid Frequency A StandardPaymentSchedule or Insurance valuable has an invalid frequency.
26 Invalid Scheduled Payments Count A Pledge cannot have over 300 scheduled payments.  The StandardPaymentSchedule installmentAmount must be increased.
27 Inappropriate Instance The database ref provided is inappropriate for the anticipated item.  For example, a valid account ref was provided for the getGift method.
28 Value Not Found A DefinedValue has a value name that doesn't exist on the field.  This can only occur for fields with a display type of single select or multi select.
29 Missing Ref A required ref was not provided.  Review the complex type being utilized to determine which refs are required.
30 Invalid Empty/Null Data An empty or null complex type was provided.  Methods with a complex type parameter cannot be null.
31 Cannot Create User Account An Account with an accountRoleType of 'User' cannot be created within eTapestry through the API.
32 Account Role Already Exists The method addAccount was called with the intention of adding a role to an existing account.  The existing account already has the accountRoleType provided.
33 Invalid Contact Method A Contact has a method that doesn't exist within the eTapestry database.
34 Missing Contact Method A Contact is missing a method.
35 Missing Contact Subject A Contact is missing a subject.
36 Missing Note A Note is missing a note.
37 Missing Field Applies To A DefinedValue cannot be assigned to an item if the field doesn't apply to that item.  Review the fault string for further details.
38 Missing Fund A transaction is mssing a fund.
39 Donor Account Required A journal entry complex type cannot be added to a user or tribute Account.  The accountRef is inappropriate.
41 Invalid Campaign A transaction has a campaign name that doesn't exist within the eTapestry database.
42 Invalid Approach A transaction has a approach name that doesn't exist within the eTapestry database.
43 Invalid Letter A transaction has a letter name that doesn't exist within the eTapestry database.
44 Invalid Credit Card Type A CreditCard has a cardType that doesn't exist within the eTapestry database.
45 Cannot Change Valuable An existing transaction cannot have its valuable changed if the transaction has been processed by the organization's transaction processor.
46 Invalid Linked Recurring Gift Schedule A RecurringGift has an invalid recurringGiftScheduleRef or recurringGiftScheduleRef is a valid RecurringGiftSchedule that is marked with a process type of auto.  A RecurringGift cannot be added to a RecurringGiftSchedule with a process type of auto.
47 Missing Real Estate Type A RealEstate is missing a realEstateType.
48 Invalid Real Estate Type A RealEstate has a realEstateType that doesn't exist within the eTapestry database.
49 Invalid Stock Shares A Stock has an invalid numberOfShares.
50 Invalid Process Valuable The method addAndProcessGift or addAndProcessPayment doesn't have a transaction with a valuable type of CreditCard or ElectronicFundsTransfer.
51 Invalid Valuable Type A Valuable provided has an invalid type.
52 Invalid Valuable A Valuable provided has a type indicated, but empty or null type data.  For example, type is 2 and check is null.
53 Invalid Non Deductible Amount A transaction has a nonDeductibleAmount that is invalid.  Review each transaction to determine what is appropriate.
54 Missing Recurring Gift Schedule Ref A RecurringGift is missing a recurringGiftScheduleRef.
55 Missing Pledge Ref A Payment is missing a pledgeRef.
56 Missing Check Number A Check valuable is missing a number.
57 Missing Credit Card Number A CreditCard valuable is missing a number.
58 Missing Stock Company A Stock valuable is missing a number.
59 Missing Insurance Insurer An Insurance valuable is missing a insurer.
60 Missing Insurance Policy Number An Insurance valuable is missing a policyNumber.
61 Missing Insurance Beneficiary An Insurance valuable is missing a beneficiary.
62 Missing Bond Maturity Date A Bond valuable is missing a maturityDate.
63 Missing Bond Issuing Entity A Bond valuable is missing an issuingEntity.
64 Invalid Phone Type A Phone has a type that doesn't exist within the eTapestry database.
66 Missing Field A DefinedValue is missing a fieldRef or fieldName.  This error can also occur when the method getUserDefinedSetValues or getSystemDefinedValues is called with a empty or null fieldName parameter.
67 Missing Value A DefinedValue is missing a value.
68 Invalid Value A DefinedValue has a value that isn't appropriate for the field's data type.  For example, the field 'Graduation Date' has a data type of date, but the value provided is '2006'.  Review the fault string for further explanation.
69 Invalid Transaction Segment Update A Gift, Payment or Pledge that belongs to a SegmentedDonation/SegmentedPledge cannot be updated by itself.  It must be updated as a segment within a SegmentedDonation/SegmentedPledge using the method updateSegmentedDonation/updateSegmentedPledge.
70 Invalid Segmented Transaction Segments A SegmentedDonation or SegmentedPledge has an invalid number of segments.
71 Invalid Segmented Transaction Segment The segments of a SegmentedDonation or SegmentedPledge has an invalid item.  Only Gifts and Payments can be segments.
72 Invalid Segmented Transaction Amount A SegmentedDonation or SegmentedPledge has a totalAmount that doesn't equal the sum of the segmented amounts or it has a totalNonDeductibleAmount that doesn't equal the sum of the segmented nonDeductibleAmounts.
73 Duplicate Segmented Transaction Segments A SegmentedDonation or SegmentedPledge cannot have two segments with the same ref.
74 Missing Soft Credit Account Ref A transaction has a softCredit that is missing an accountRef.
75 Invalid Auto Process Valuable A Pledge or a RecurringGiftSchedule having a schedule marked with a process type of auto has an empty/null valuable or a valuable other than CreditCard or ElectronicFundsTransfer.
76 Invalid Process Type A StandardPaymentSchedule or CustomPaymentSchedule has an invalid processType.
77 Missing First Installment Date A StandardPaymentSchedule is missing a firstInstallmentDate.
78 Missing Date A journal entry complex type is missing a date.
79 Missing Phone Type A Phone is missing a type.
80 Missing Persona Type An Account is missing a personaType.
81 Invalid Country Code An Account has a country that doesn't exist within the eTapestry database.
82 Invalid Custom Schedule Amounts A CustomPaymentSchedule has an empty or null installmentAmounts array.
83 Invalid Custom Schedule Dates A CustomPaymentSchedule has an empty or null installmentDates array.
84 Invalid Custom Schedule Details A CustomPaymentSchedule has a installmentAmounts or installmentDates array that has only one entry or the two arrays don't have the same length.
85 Invalid Custom Schedule Date A CustomPaymentSchedule has a installmentDates array entry with a missing date.
86 Missing Account Ref A journal entry complex type is missing an accountRef.
87 Invalid User Defined Field Display Type A DefinedValue has an invalid displayType.
88 Invalid User Defined Field Data Type A DefinedValue has an invalid dataType.
89 Invalid EFT Account Type An ElectronicFundsTransfer has an invalid accountType.
90 Invalid Recognition Type An Account has an invalid donorRecognitionType or a transaction has an invalid recognitionType.
91 Invalid Account Role Type An Account has an invalid accountRoleType.
92 Missing Payment Schedule A Pledge is missing a standardSchedule or customSchedule.
93 Field is not a Set The field provided for method getUserDefinedSetValues or getSystemDefinedValues is not a set (i.e. display type single select or multi select).
94 Invalid Id The id (aka Account Number) provided is invaild.  This number must be greater than zero.
95 Missing Search Criteria No search criteria was provided within AccountSearch.
96 Invalid Account Search Role Types An AccountSearch has an invalid accountRoleTypes value.
97 Invalid Payment Pledge Ref A Payment has a pledgeRef that isn't a pledge transaction.
98 Invalid Payment Addition A Payment has a pledge indicated (pledgeRef) that cannot have a payment added to it.  Payments cannot be added to pledges that are written off, reversed, or fulfilled.
99 Invalid Matching Refs A complex type was provided with multiple ref values populated with the same value.  Two or more of those same value refs cannot have matching refs.
100 Missing Duplicate Search Criteria Search criteria was missing within DuplicateAccountSearch.  Consult getDuplicateAccount to learn what fields are required.
101 Missing Required Parameter A required method parameter was provided a null or empty value.
102 Invalid Query Category A query category name provided is invalid.
103 Invalid Query A query name provided is invalid.
104 Donor Login Role Not Allowed The account with the provided login credentials doesn't have access to use the api.
105 Temporarily Unavailable The system is temporarily unavailable.  Please check back shortly.
106 Inappropriate Processor The organization's transaction processor doesn't support this action.
107 Invalid Transaction Update The transaction update attempted isn't allowed.  Review the fault string for further explanation.
108 Cannot Change Amount The transaction amount cannot be changed on a transaction that has been processed.
109 Missing Valuable The transaction is missing a valuable.
110 Invalid Reverse Transaction The transaction reversal attempted isn't allowed.  Review the fault string for further explanation.
113 Cannot Update Immutable Entry Journal Entries marked as immutable cannot be updated.
114 Login Not Enabled The account with the provided login credentials obtains a disabled login id.
115 Invalid Login Time The account with the provided login credentials doesn't have access to the database at the current time.
