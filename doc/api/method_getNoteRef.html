getNoteRef Method
Order Name Type
Return noteRef string
Input 1 definedValue DefinedValue

Comments
1) The purpose of this method is to locate a single Note with the definedValue provided.  If multiple notes have the definedValue provided, soap fault error number 7 will occur.  The value traditionally provided is a unique id that is stored within a display type text field.
2) A soap fault will occur if the parameter definedValue is empty or invalid.  Review DefinedValue to help avoid a soap fault.
