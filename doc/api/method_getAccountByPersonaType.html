getAccountByPersonaType Method
Order Name Type
Return account Account
Input 1 accountRef string
Input 2 personaType string
Input 3 returnEmptyPersonaShell boolean

Comments
1) A soap fault will occur if the parameter accountRef or personaType are empty or invalid.
2) If parameter returnEmptyPersonaShell is true and the account doesn't have the the personaType provided, all Account fields will be populated appropriately except the Account fields with special comment 5.
