addDisbursement Method
Order Name Type
Return transactionRef string
Input 1 disbursement Disbursement
Input 2 createFieldAndValues boolean

Comments
1) This method will add a new disbursement transaction to an existing account.
2) A soap fault will occur if the parameter disbursement is empty or invalid.  Review Disbursement to help avoid a soap fault.
3) If the parameter createFieldAndValues is true, the following items will be created if they aren't located: fund, campaign, approach, letter, and all DefinedValue fields and values.  When this parameter is set to true, don't forget to set each DefinedValue dataType and displayType to appropriate values.
