getAccountByUniqueDefinedValue Method
Order Name Type
Return account Account
Input 1 definedValue DefinedValue

Comments
1) The purpose of this method is to locate a single Account with the definedValue provided.  If multiple accounts have the definedValue provided, a soap fault (7 - Duplicate Matches Found) will occur.  The value traditionally provided is a unique id that is stored within a display type text field.
2) A soap fault will occur if the parameter definedValue is empty or invalid.  Review DefinedValue to help avoid a soap fault.
