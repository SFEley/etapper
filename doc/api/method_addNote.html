addNote Method
Order Name Type
Return noteRef string
Input 1 note Note
Input 2 createFieldAndValues boolean

Comments
1) This method will add a new journal note to an existing account.
2) A soap fault will occur if the parameter note is empty or invalid.  Review Note to help avoid a soap fault.
3) If the parameter createFieldAndValues is true, this indicates all DefinedValue fields and values that aren't located will be created.  When this parameter is set to true, don't forget to set each DefinedValue dataType and displayType to appropriate values.
