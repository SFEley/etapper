I received the following soap fault. How do I avoid it?
This normally means you have an int or boolean variable on a complex type that was passed as null (e.g. xsi:nil='true'). Assuming you are using proxy objects, try initializing int variables to zero and boolean variables to false. This should prevent the variable from being null.
