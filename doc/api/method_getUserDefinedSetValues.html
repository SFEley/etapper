getUserDefinedSetValues Method
Order Name Type
Return values DefinedValue array
Input 1 fieldName string
Input 2 includeDisabled boolean

Comments
1) A soap fault will occur if the parameter fieldName is empty or invalid.
2) Only fields with a display type of single select or multi select are allowed.
3) Individual User Defined Fields can be located within eTapestry under Manage...Defined Fields...User Defined Fields.
4) A high level view of User Defined Fields can be obtained by running one of the User Defined Field Reports within the Main...Report/Export category of 'eTapestry Standard Reports'.
